Abstract
Ce document présente les ressources spécialisées en sciences et techniques pertinentes pour le cours de recherche dans les sources spécialisées (7V46-SRI-1) donné dans la filière Information Science de la Haute école de gestion de Genève.

1 Recherche d’information dans les sources spécialisées en sciences & techniques

1.1 Contexte

Le cours Recherche d’information dans les sources spécialisées (HEG-7V46-SRI-1) dispensé à la Haute école de gestion (HEG) de Genève, filière Information Science, est découpé en 3 parties, couvrant chacune des domaines différents:

droit, économie et gestion
sciences humaines et sociales
sciences et techniques

IMPORTANT
Ces notes ne concernent que les ressources en sciences et techniques.

1.2 Objectifs du cours

Cf. descriptif du module 7V4/6-SRI

Objectifs pédagogiques :
* Connaître les spécificités de la recherche d’information par domaine du savoir et les principales sources utilisées ;
* savoir utiliser ces sources ;
* être capable de sélectionner ces sources en fornction des besoins d’informations ;
* savoir citer les références trouvées.
